Questions

  1. The server in the client server architecture wait for requests from the clients and responds. For instance I as a client would request a web page and the server responds with the web page that I requested.
  2. The client request and recieve service and information from the server.
  3. An IP, short for internett protocol, is a unique numeric identifier that every computer on the Internett has. Through this we can distinguish the different computers connected to the internet, and with different IP adresses the client could be directed to the right web page. If the numbers weren't unique the client could be directed to the wrong web page. The protocol guarantees proper transmission of data.
  4. DNS, short for Domain Name Server, is what "translates" IP address' to homepage names. It is a server that connects the IP address to the homepage names.
  5. The Internett is a much broader term. The internet contains much more than that of WWW (the World Wide Web). The internet is a network of networks. The World Wide Web is a system of hypertext documents that is accessed over the Internet.
  6. HTML, short for Hypertext Markup Language, is a language that is used by WWW. HTML is a standard language that is used to describe how the web page is formatted. What the web page will look like.
