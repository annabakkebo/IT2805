Questions - first assignment

First assignment
  1. The server in the client server architecture wait for requests from the clients and responds. For instance I as a client would request a web page and the server responds with the web page that I requested.
  2. The client request and recieve service and information from the server.
  3. An IP, short for internett protocol, is a unique numeric identifier that every computer on the Internett has. Through this we can distinguish the different computers connected to the internet, and with different IP adresses the client could be directed to the right web page. If the numbers weren't unique the client could be directed to the wrong web page. The protocol guarantees proper transmission of data.
  4. DNS, short for Domain Name Server, is what "translates" IP address' to homepage names. It is a server that connects the IP address to the homepage names.
  5. The Internett is a much broader term. The internet contains much more than that of WWW (the World Wide Web). The internet is a network of networks. The World Wide Web is a system of hypertext documents that is accessed over the Internet.
  6. HTML, short for Hypertext Markup Language, is a language that is used by WWW. HTML is a standard language that is used to describe how the web page is formatted. What the web page will look like.

Questions - second assignment

  1. Why does inline style CSS override rules defined in style elements and external stylesheets?
    • When building the HTML file with CSS it always uses the last defined style. The inline style CSS is then the last defined and then override the rules defined in the style element or external stylesheets. This way you could use the defined style elements or external stylesheets as a general template for the web page. When changing the style of spesific elements on the page the inline style CSS would override.
  2. Give a brief example of when to use ID (#id) and when to use classes (.class) in CSS.
    • ID is used when identifying one element, while classes can be used when identifying several elements.
  3. What does RGBA mean and what colors can you express with it?
    • RGBA, red-green-blue-alpha, defines what intensity of red, green or blue the color will have. In addition it defines the opacity of the color as the alpha input. The red, green and blue inputs are integers between 0 and 255 or percentage. Alpha is given as a number between 0 and 1. Where 0 is transparent and 1 is visible.
      We can express all colors with it.
  4. Why do we include CSS files inside the head element and not inside the body element?
    • Because then the web page has the styles before starting to ''paint'' the web page. If it had been inside the body element the web page would start to load the page, and then recognize the CSS file / style element. It would then need to reload the page to implement the style.
  5. What CSS selector matches all the p elements inside the article element in the following HTML?
    <p> This should not match </p>
    <article>
    <h2>This should not match</h2>
    <p>This should match.</p>
    <p>This should also match.</p>
    <p>Do not forget about this one!</p>
    </article>
